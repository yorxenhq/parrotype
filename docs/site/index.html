## how it works

No windows to click, nothing steals focus. A small pill above the taskbar shows what's happening — that's the whole UI.

01

Hold Ctrl + Alt

Push-to-talk, anywhere in Windows. Prefer a toggle? Ctrl+Shift+Space starts and stops.

02

Speak

A live waveform shows the mic is hearing you. Esc cancels — nothing gets inserted.

03

Release — text lands

Whisper transcribes locally in ~0.8s and the text is typed right where your cursor is. Clipboard untouched.

## private by design

Not "privacy-friendly". Offline, verifiably.

  • Recognition runs on your machine. OpenAI Whisper via faster-whisper (CTranslate2). Your voice and text never leave it.
  • The dictation path contains no network code. Don't trust us — grep the source.
  • The network is touched only when you ask: the one-time model download at setup, the optional GPU-runtime download behind its own button, and a weekly update check you can switch off.

Types directly

EN, RU, ES, DE, FR, IT, PT, PL, UK, NL, TR, JA, KO, ZH — each passed our own accuracy test. Auto-detect for 90+.

## faq

Does it work offline?

Yes. The model downloads once on first run — ~75 MB to ~3 GB, you pick the size. After that, recognition is fully local; airplane mode is fine. The only other network call is an optional weekly check for a new version, and you can switch it off in Settings.

Which languages does it understand?

English, Russian, Spanish, German, French, Italian, Portuguese, Polish, Ukrainian, Dutch, Turkish, Japanese, Korean and Chinese — each passed a measured recognition-quality gate before it made the menu. Auto-detect covers the 90+ languages Whisper knows. The interface itself speaks English and Russian.

Where does my data go?

Nowhere. Audio is processed in memory and transcribed on your machine. Dictation history, if you keep it on, is a local file capped at the last 50 entries — you can clear it or turn it off. No accounts, no telemetry, no analytics.

Windows SmartScreen warns about it — why?

The installer isn't code-signed yet: certificates cost real money and Parrotype is free. Click "More info", then "Run anyway". If you'd rather trust the code than us — it's open source, build it yourself in four commands.

How does it update?

It doesn't touch anything by itself. Once a week it can ask GitHub whether a new version is out — one anonymous request, nothing about you is sent, and you can turn it off. Updating is just downloading the new installer.

Do I need a GPU?

No. On a regular CPU a phrase takes a couple of seconds with the default model. Have an NVIDIA card? The app offers a one-click CUDA download and the best model answers in ~0.8 s per phrase, measured.
