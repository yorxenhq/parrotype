# Parrotype

**You talk. The parrot types.**

Local voice dictation for Windows: press a hotkey, speak, release — your words are typed into whatever window you're in. Runs fully on your machine (Whisper-based). No cloud, no account, no audio leaving your laptop.

Status: pre-build. Spec finalized, first build pending.

## Planned

- Push-to-talk + toggle modes, global hotkey
- Mixed-language dictation (RU/EN and friends)
- Live waveform overlay, tray control, replacement dictionary
- Core engine separated from UI — embeddable (CLI / local HTTP)
